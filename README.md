# Continuum

Website for Continuum — Style Stewardship by Kemissa.

## Tech

- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui

## Local development

```sh
npm install
npm run dev
```

Dev server runs on `http://localhost:8080`.

## Build

```sh
npm run build
```

Output goes to `dist/`.

## Hosting

Deployed on Cloudflare Pages from the `main` branch of this repo. Build command `npm run build`, output directory `dist/`.

## Contributor workflow

See `CHANGELOG.md`. Two contributors work on this repo — `git pull --rebase` before starting, and log a short entry under **Log** before pushing.
